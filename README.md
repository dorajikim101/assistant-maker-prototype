# Assistant Maker Prototype

A lightweight Princess Maker-inspired onboarding prototype for a persistent AI assistant.

## Included screens
- Birth / role setup
- Library / book-based preference inference
- Training track selection
- Ending / export preview

## Local preview
```bash
python3 -m http.server 4173
```
Then open <http://127.0.0.1:4173>.

## Deploy to GitHub Pages
This repo is static HTML/CSS/JS and works directly on GitHub Pages.
